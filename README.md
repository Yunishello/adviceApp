### What I learned

I found it interesting to finally know how to use this amazing css trick on how to position an element.

```css

  position: absolute;
  top: 100%;
  left: 50%;
  -ms-transform: translate(-50%, -50%);
  transform: translate(-50%, -50%);
}
```
